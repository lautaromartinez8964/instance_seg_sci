# 面向端侧部署的密集砾石几何驱动实例分割：完整学术研究手册

## 摘要（论文一句话核心叙事）

本工作针对手机端密集土壤砾石实例分割任务中**尺度极端分化下的微小目标消失、密集粘连下的实例边界消失、以及背景高度异构下的语义混淆**三大根本难题，提出一种**几何驱动的轻量分割框架**。该框架跳脱传统逐像素语义分类范式，通过**(1) 极坐标自适应实例建模**从物理表示层面消除小目标的下采样消失问题，**(2) 实例间对比学习边界感知**从特征学习层面强制邻近实例的特征分离，以及**(3) 不确定性感知的SAM几何蒸馏**从知识迁移层面将基础模型的几何理解无损注入端侧网络。整套方案推理时计算开销极低，专为手机实时分割设计。

---

## 1. 任务定义与核心挑战

### 1.1 任务定义

-   **输入**：手机在田间自然光下拍摄的高分辨率RGB图像（典型分辨率640×640 – 3012×3012，自适应处理）。
-   **输出**：图中每一颗砾石的实例级分割掩码（单类“gravel”），无需粒径测算。
-   **约束**：模型需在端侧CPU/NPU上实现实时或近实时推理（单图<100ms），参数量<10M。

### 1.2 三大场景特异性难题（本工作的唯一攻关对象）

| 问题 | 物理表现 | 现有方案的致命缺陷 |
| :--- | :--- | :--- |
| **P1: 尺度极端分化** | 同一图中有直径>200px的大砾石和<12px的微小砾石 | 笛卡尔固定尺寸掩码在下采样后微小砾石仅剩0-1个特征点，“结构性消失”；且大掩码对小目标存在严重参数冗余 |
| **P2: 密集粘连，大压小** | 大砾石紧挨小砾石，图面几乎无纯背景，相邻实例仅靠极窄缝隙/阴影区分 | 标准NMS后处理会抑制与高置信度框高度重叠的小目标；通用FPN跨层融合会将相邻实例的特征混合，使边界不可分 |
| **P3: 背景选择性伪装** | 土壤类型多样，某些土质颜色、纹理与砾石表面高度相似；甚至在纯砾石场景中，实例间“背景”本身就是砾石 | 仅依赖RGB纹理的外观模型在该场景下大面积失效，缺乏互补的几何判别线索 |

**根本矛盾：** 传统实例分割模型本质是“语义驱动的像素分类器”，而砾石是一类**几何属性（轮廓、缝隙、形状）远强于语义属性**的目标。这三重困难共同指向一个需求：**我们需要一个“几何驱动的形状理解器”，而非更深的像素分类器。**

---

## 2. 整体框架：基础支柱与三大创新

### 2.1 基础模型骨架

采用**RTMDet-Ins (tiny)** 作为一阶段轻量检测分割基座。该基座已集成两项已初步验证的场景增强组件，但不作为本文核心创新：

-   **Shared-DT DT-FPN**（用于提供粗略的实例中心-边界空间先验，保证全局几何梯度）
-   **NWD-based Dynamic Soft Label Assigner**（用于提升密集小目标的训练分配稳定性，替换默认IoU）

> 以上两项为本文的**工作基础**，确保了基线模型在密集小目标场景下的最低可用性。本文的三大核心创新将在此基础上实现质的跃升。

### 2.2 三大创新点逻辑链

-   **创新一（表示层）**：用极坐标自适应建模**根本性解决小目标的表示与消失问题**。
-   **创新二（特征层）**：用实例间对比学习**强制模型在特征空间推开邻近实例，实现边界分离**。
-   **创新三（知识层）**：用不确定性感知的SAM蒸馏**将大模型的精细几何知识迁移到端侧轻量网络**。

三者环环相扣：极坐标头提供了几何表达的能力，对比学习使特征空间具备几何判别力，而SAM蒸馏则为两者注入了超越人工标注的精细结构监督。

## 3. 创新点一：极坐标自适应实例建模

### 3.1 针对问题

**P1（尺度极端分化）与 P2（密集粘连中的小目标漏检）**

### 3.2 核心思想

将实例掩码的表示从固定分辨率的笛卡尔网格，转变为尺度无关的参数化闭合轮廓。每个实例仅用一个中心点和一组发射状射线长度来表达，使小目标的几何表达不再受制于像素网格下采样。

### 3.3 具体实现

在RTMDet-Ins的检测头中，**并排添加**一个极坐标掩码头（PolarMaskHead），与原有的框回归头协同工作。

1.  **实例中心与复杂度预测**：
    -   对于每个候选框，RoI Align提取14×14特征。
    -   一个极小的全卷积分支预测**轮廓复杂度分数** \(\gamma \in [0,1]\)。
    -   该分数决定本实例的射线数量 \(N = N_{min} + \gamma \cdot (N_{max} - N_{min})\)，默认 \(N_{min}=32, N_{max}=128\)。

2.  **自适应射线长度回归**：
    -   在 \(N\) 个等角度方向 \(\theta_k = \frac{2\pi k}{N}\) 上，预测射线长度向量 \(\mathbf{r} = [r_1, ..., r_N]\)。
    -   射线原点是框中心。

3.  **掩码重建与损失**：
    -   由\(\mathbf{r}\)重建多边形，渲染为分割掩码。
    -   **掩码损失** \(L_{mask}\)：使用自适应加权的Dice Loss + Focal Loss。
    -   **复杂度正则损失** \(L_{comp} = \lambda_\gamma \cdot \gamma\)，鼓励网络用小射线数描述简单圆形砾石。
    -   检测框的质量仍由NWD Assigner筛选的正负样本和定位损失监督，极坐标头只负责轮廓精修。

**学术贡献点：** 首次在密集颗粒实例分割中引入自适应几何预算的轮廓建模，从物理表示层面消灭了小目标的下采样消失问题，且天然避免了笛卡尔掩码的参数冗余。

## 4. 创新点二：实例间对比学习边界感知

### 4.1 针对问题

**P2（密集粘连）与 P3（背景伪装/全砾石场景下边界不可见）**

### 4.2 为什么不用显式的边界/缝隙预测

我们前期的探索发现，强迫网络显式预测一个1-2像素宽的“接缝线”图（IBD），并用它硬门控特征，在实践中几乎必然失败。原因：
-   接缝线预测本身就是和最终分割等难甚至更难的任务，存在鸡生蛋的循环依赖。
-   硬门控对IBD预测误差零容忍，错误信号直接杀死正确特征。
-   在全砾石图像中，无数表面纹理产生高频噪声，无明确方向的频域滤波无法干净分离缝隙。

因此，我们完全摒弃“显式预测→门控”路线，转向**从实例标注中直接构造的、作用于特征空间的软约束**。

### 4.3 方案：面向边界的实例间对比学习

**目标：** 不强制网络画边界，而是要求它在特征空间中做到：**同一实例内的像素特征互相靠近，空间相邻但属于不同实例的像素特征互相远离。**

**实现细节：**

-   **采样策略（在线进行）**
    从GT实例掩码中，每次训练迭代随机采样若干组像素对：
    1.  **正样本对 \(P\)**：来自同一实例，且空间距离大于实例直径的1/3（避免只约束邻近像素，强制实例内全局一致性）。
    2.  **难负样本对 \(N_{hard}\)**：来自两个不同实例，但空间欧氏距离小于12像素（物理上处于“接缝”两侧）。这是核心监督源。
    3.  **易负样本对 \(N_{easy}\)**：来自不同实例且距离>50像素，用于维持特征空间宏观结构。

-   **特征提取**
    在FPN的某一中等尺度层（如P3）输出上，采样对应位置的特征向量。

-   **对比损失函数**
    \[
    L_{contrast} = -\frac{1}{|P|}\sum_{(i,j)\in P}\log\frac{e^{sim(f_i,f_j)/\tau}}{\sum_{k}...}  \quad \text{(简化: 使用余弦相似度与margin)}
    \]
    更稳定的形式是：
    \[
    L_{bound} = \frac{1}{|N_{hard}|} \sum_{(i,j)\in N_{hard}} \max(0, \, m - \|f_i - f_j\|_2)
    \]
    其中\(m\)为设定的分离margin。对于正样本对，可使用L2距离损失以聚拢特征。

-   **与检测头的协同**
    该损失作为辅助损失，直接加入总目标。它不给特征图增加任何额外的预测头，仅在训练阶段促使FPN输出的特征在实例边界处“自发”产生不连续性。受益方包括后续的极坐标头（更容易找到轮廓终点）以及框回归分支（特征更聚拢）。

**学术贡献点：** 提出一种面向密集实例分割边界的对比学习框架，首次将“推开邻近实例特征”的思想引入轻量分割器，完全避免了显式边界预测的噪声敏感和循环依赖，以一种极其稳健的方式赋予了模型实例间特征分离的能力。

## 5. 创新点三：不确定性感知的SAM几何蒸馏

### 5.1 针对问题

**人工标注边界粗糙、轻量模型几何感知能力受限**

我们的端侧轻量网络受限于参数量和感受野，无法像大基础模型那样平滑地感知砾石的几何连续结构。直接的掩码蒸馏（学习SAM二值分割图）是粗暴的——它无视了SAM在砾石粘连区域自身的不确定性，且只是复制答案，未传递几何理解。

### 5.2 方案：蒸馏结构知识，而非输出标签

我们提出一种**不确定性感知的几何结构蒸馏**策略，仅在训练时使用服务器端SAM2，推理零开销。

#### 5.2.1 离线知识生成

对训练集每一张图，用SAM2以**多种不同提示（如3×3网格点+随机单点多次）** 运行，得到：
-   一组二值掩码 \(\{M^k\}\)。
-   **教师距离变换图 \(D_{tea}\)**：由所有\(M^k\)的均值掩码计算。
-   **教师边界置信度图 \(B_{tea}\)**：由各\(M^k\)边界频率得出，连续值∈[0,1]。
-   **不确定性权重图 \(W_{uncer}\)**：基于多个\(M^k\)的逐像素方差和边界抖动计算，在教师自己也犹豫的区域数值趋近于0（如极度粘连的细小接缝处），在确定区域趋近于1。

以上数据全部离线保存为`.npy`，不占用训练GPU时间。

#### 5.2.2 学生端几何蒸馏头

在轻量学生的FPN深层，附加一个极小的**几何解码器**（约0.05M参数）：
-   输出学生距离变换图 \(D_{stu}\)。
-   输出学生边界置信度图 \(B_{stu}\)。

该解码器**仅在训练时激活**，推理阶段被删除。

#### 5.2.3 不确定性加权蒸馏损失

\[
L_{geo\_distill} = \lambda_D \cdot W_{uncer} \cdot \text{MSE}(D_{stu}, D_{tea}) + \lambda_B \cdot W_{uncer} \cdot \text{BCE}(B_{stu}, B_{tea})
\]

-   在SAM自己都不确定的地方（\(W_{uncer}\)→0），损失归零，学生此时完全信赖人工GT和自身特征学习。
-   在SAM高确定的区域，学生则学习教师平滑、精细的几何结构。

#### 5.2.4 渐进式课程蒸馏

-   **Phase 1（主干期, 0-60% iterations）**：仅使用人工GT训练全部结构，包括极坐标头、对比学习损失。学生先形成基本分割能力。
-   **Phase 2（精炼期, 60-100% iterations）**：逐步加入\(L_{geo\_distill}\)，权重由0线性增加到最终值。让学生在用教师高阶知识精调时不致迷失。

**学术贡献点：** 设计了一种专为密集几何场景定制的、不确定性感知的知识蒸馏范式。它首次实现了从基础模型到端侧专用网络的“结构直觉”传递，而非以往的输出标签压缩。不确定性加权机制赋予了蒸馏过程自我校正的能力，避免教师错误固化为学生偏见。

## 6. 整体训练管线与超参数建议

### 6.1 数据准备

-   **数据集**：自建2000张手机拍摄砾石图，全部使用XAnyLabeling进行精细多边形全标注。训练/验证/测试=1400/300/300.
-   **数据增强**：随机翻转旋转、多尺度缩放、亮度对比度色相扰动；针对密集场景加入Mosaic拼接。

### 6.2 模型架构总览

```
输入
  │
  ▼
CSPNeXt Tiny Backbone
  │
  ▼
Shared-DT DT-FPN (已有)  ← 提供基础几何先验
  │
  ├─────────────── 对比学习损失 (创新二)
  │                (从P3层采样，推开难负样本)
  ▼
NWD Dynamic Assigner (已有) + 框回归头
  │
  ▼
自适应极坐标掩码头 (创新一) → 轮廓射线预测
  │
  ▼
[仅训练时] 几何蒸馏解码器 ← SAM2教师监督 (创新三)
```

### 6.3 损失函数

\[
L_{total} = \underbrace{L_{det} + L_{polar\_mask} + \lambda_c L_{comp}}_{基础任务损失} + \underbrace{\lambda_b L_{bound\_contrast}}_{创新二} + \underbrace{\lambda_g \cdot s(t) \cdot L_{geo\_distill}}_{创新三(渐进式)}
\]

其中\(s(t)\)是随训练迭代从0到1的线性函数。

## 7. 实验设计重点

### 7.1 核心对比指标
-   **AP, \(AP_{small}\)（小目标）, \(AP_{medium}\), \(AP_{large}\)**：验证对尺度极端变化的适应性。
-   **\(AP_{boundary}\)**：在实例轮廓2像素窄带内计算的分割精度，是证明分离能力的最关键指标。
-   **参数量, FLOPs, 端侧推理延迟(FPS)**：证明方案完全满足部署约束。

### 7.2 消融实验矩阵
1.  基线 (RTMDet-Ins + DT-FPN + NWD)
2.  基线 + 极坐标头 (创新一)
3.  基线 + 对比学习 (创新二)
4.  基线 + 极坐标头 + 对比学习
5.  完整模型 (+ 创新三)
6.  蒸馏的确定性加权消融（有无不确定性加权、有无两阶段课程）

### 7.3 定性可视化
-   极坐标射线可视化，展示大石头多射线、小石头少射线的自适应效果。
-   特征图cosine距离图，展示对比学习使边界处特征不连续性显著增强。
-   与SAM、YOLO-Seg、传统笛卡尔头的分割结果特写对比，聚焦粘连区域的撕裂效果。

## 8. 核心贡献陈述（可直接用于摘要）

1.  **We introduce an adaptive polar instance modeling scheme**, which replaces fixed Cartesian masks with a scale-invariant contour representation, eliminating the small-instance vanishing problem and enabling adaptive geometric complexity allocation.
2.  **We design an inter-instance contrastive boundary-aware learning paradigm**, which forces the feature encoder to separate spatially adjacent but distinct instances without requiring any explicit edge prediction, providing robust boundary discrimination under dense clutter.
3.  **We develop an uncertainty-aware SAM geometric distillation framework**, which transfers fine-grained structural knowledge (distance fields and boundary confidence) from a large foundation model to the lightweight on-device network, equipped with a self-correcting mechanism that prevents the inheritance of teacher uncertainties.
4.  These three geometry-driven innovations collectively elevate a lightweight one-stage detector into a shape-aware instance segmenter specifically tailored for dense gravel scenes, achieving expert-level boundary separation under strict on-device constraints.
