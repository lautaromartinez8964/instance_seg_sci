### 📷 拍摄技术规范

#### **设备要求**
| 设备 | 推荐 | 备用 |
|-----|------|------|
| **主设备** | 智能手机（iPhone 12+/华为P40+/小米11+） | - |
| **辅助设备** | 三脚架（带横臂，可垂直俯拍） | - |
| **比例尺** | 30cm直尺或1元硬币（直径25mm） | 必须每张都有 |

#### **拍摄参数**
| 参数 | 要求 |
|-----|------|
| **拍摄角度** | 垂直俯拍（90°±5°），使用手机水平仪校准 |
| **拍摄高度** | 50cm固定高度（用三脚架横臂或自拍杆） |
| **分辨率** | 最高分辨率（≥3024×4032） |
| **对焦** | 手动对焦到地面中心，确保砾石清晰 |
| **曝光** | 自动曝光，避免过曝/欠曝 |
| **格式** | JPG（高质量）或 RAW（如有） |

#### **拍摄流程**
1. **放置比例尺**：在拍摄区域边缘放直尺或硬币
2. **架设三脚架**：调整到50cm高度，确保水平
3. **构图**：取景框覆盖约40×30cm地面
4. **对焦**：点击屏幕中心对焦
5. **拍摄**：连拍3张（防止手抖）
6. **检查**：放大查看砾石是否清晰

```
B. GravelSeg-XZ Dataset

We construct GravelSeg-XZ, a high-resolution gravel instance segmentation 
dataset captured in Xuzhou, Jiangsu Province, China. The dataset contains 
770 raw images (3024×4032) captured across 18 diverse geological sites, 
including Yunlong Lake shorelines, Yulong Mountain weathered rocks, mining 
areas near China University of Mining and Technology, and the ancient Yellow 
River bed. After cropping with 50% overlap, we obtain 4,200 image patches 
at 1024×1024 resolution.

Each image is annotated with pixel-level instance masks for gravel particles, 
categorized into three size classes based on equivalent diameter: coarse 
(>20mm), medium (5-20mm), and fine (<5mm). The dataset covers varying 
illumination conditions (sunny, cloudy, post-rain), ground moisture levels 
(dry, moist, wet), and gravel densities (5%-80% coverage). To our knowledge, 
this is the first public dataset specifically designed for gravel segmentation 
in soil science and geological survey applications.
```
