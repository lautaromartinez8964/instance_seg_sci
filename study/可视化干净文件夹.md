## 2. iSAID 的 `val/images` 目录本来就可能包含三类文件
你仓库 README 里写得很清楚，iSAID patch 目录下会出现：

- `P0003_0_0_800_800_instance_id_RGB.png`



### 实例 ID 图
```text

```

如果你直接把整个 `val/images` 文件夹喂给 `image_demo.py`，它会把这三类图都当“待推理图片”处理。  
这就会造成你现在看到的混乱现象。
